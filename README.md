# Avi-Cenna Admin Dashboard

A lightweight, serverless administrative interface for managing blog posts, news, and academic updates. This dashboard connects a vanilla HTML/JS frontend to a Google Sheets database via Google Apps Script (GAS).

## 🚀 Features
- **Secure Authentication**: Credentials-based access to the dashboard.
- **CRUD Operations**: Create, read, and delete blog posts in real-time.
- **Rich Content Support**: Basic HTML formatting for post content.
- **Cloud Database**: Powered by Google Sheets for easy data management.
- **Session Persistence**: Stays logged in using browser local storage.
